Teaching Tools#

I develop open source software tools designed to improve teaching and learning in physics and computational science. These tools are freely available for anyone to use, adapt, and build upon.

Each tool is designed with educators and students in mind, emphasizing accessibility, ease of use, and pedagogical effectiveness. Below are the tools I maintain:
